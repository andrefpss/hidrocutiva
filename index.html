Nosso projeto de irrigação automática é uma solução intelegente para o sistema agrario. Nele é ultilizado o sensor de humidade O sensor de umidade é projetado para ser inserido no solo, onde monitora constantemente os níveis de umidade ao redor das raízes das plantas. Com tecnologia de ponta, ele detecta com precisão quando o solo está ficando seco, sinalizando ao sistema de irrigação para entrar em ação.
